# Features
+ Job Application Form. 
   -  A Semantice Rect-ui design form for applicants to submit their candidature details.
   -  Applied custom validation
   -  Displayed apropriated error message
+ Employer DashBoard 
   -  A Semantic React ui admin panel for review of applicants.
   -  Listed all users in the table form user ease to view data.
   -  Each of the users applicats form deatails are displyed in the tabs depending on the choice of job 
   -  Employer can view individuals details in a backdrop using sweetalert.
   -  Employer can mark the candidates as accepted or rejected after review.
